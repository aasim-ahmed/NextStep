## ✨ Features

- 🤖 **AI-Powered Roadmaps** — Generates structured, step-by-step learning paths using Google Gemini AI
- 📚 **Curated Resources** — Returns books, courses, YouTube channels, and websites tailored to your topic
- 💼 **Career Insights** — Provides top hiring companies, in-demand roles, and hiring trends
- 💰 **Salary Data** — Compares average packages across India, USA, and Europe
- 💻 **Project Ideas** — Suggests hands-on projects with difficulty levels and GitHub examples
- 🔮 **Future Trends** — Analyzes market demand and emerging technologies in your field
- 📂 **Guide History** — Saves all past guides to your profile; view any previous guide from the sidebar
- 🔐 **JWT Authentication** — Secure login and registration with JSON Web Tokens
- 🎨 **Modern Dark UI** — Premium SaaS-style interface with glassmorphism and micro-animations

## 🏗️ Architecture

```
Browser (React SPA)
       │
       │  HTTP REST (Axios)
       ▼
Express.js API (Port 5500)
       │
       ├── /api/auth  ──────────────► Auth Controller ──► MongoDB (Users)
       │
       ├── /api/user  ──────────────► User Routes ──────► MongoDB (Users + populated Guides)
       │         [authMiddleware]
       │
       └── /api/ai    ──────────────► AI Route ──────────► Google Gemini AI
                 [authMiddleware]                               │
                                                               ▼
                                                     MongoDB (CareerGuides)
```

- The frontend is a **single-page React application** served by Vite.
- All API requests from the frontend are authenticated via a **JWT token** stored in `localStorage`.
- The `authMiddleware` verifies the token before allowing access to protected routes and injects `req.user` securely.
- The AI guide is generated by calling **Google Gemini** with a structured prompt and strict JSON output format, then persisted to MongoDB and linked to the user's `chats` array.

## ⚙️ Installation

### Prerequisites
- **Node.js** v18+
- **MongoDB** (local or [MongoDB Atlas](https://cloud.mongodb.com))
- **Google Gemini API Key** (get one at [Google AI Studio](https://aistudio.google.com/app/apikey))

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/NextStep.git
cd NextStep
```

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file in the `backend/` directory:

```env
PORT=5500
MONGO_URI=mongodb://localhost:27017/NextStep
NODE_ENV=development
GEMINI_API_KEY=your_google_gemini_api_key_here
JWT_SECRET=your_strong_jwt_secret_here
```

Start the backend server:

```bash
npm start
# or for hot-reload during development:
npm run dev
```

### 3. Frontend Setup

```bash
cd ../frontend
npm install
npm run dev
```

The frontend will be available at **http://localhost:5173**

## 🚀 Usage

1. **Sign up / Log in** at `/auth` to create your account.
2. Navigate to **My Guides** (`/guide`) from the navbar.
3. Fill in your **name**, **study topic**, and **current experience level**.
4. Click **Generate My Personalized Guide** and wait for the AI to craft your roadmap.
5. Your guide includes:
   - A structured **5-step learning roadmap**
   - **Books, courses, YouTube channels, and websites**
   - **Project ideas** with difficulty ratings
   - **Top hiring companies** and career trends
   - **Salary comparisons** across regions
6. All your past guides are saved and accessible from the **Sidebar**.

## 📡 API Endpoints

### Authentication — `/api/auth`
| Method | Endpoint | Body | Description |
|--------|----------|------|-------------|
| `POST` | `/register` | `{ fullName, avatar, email, password }` | Register a new user |
| `POST` | `/login` | `{ email, password }` | Login and receive a JWT token |

> All routes below require the `Authorization` header: `Authorization: <token>`

### User — `/api/user`
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/previousChats/:userId` | Fetch all past guides for the authenticated user |
| `PUT` | `/update` | Update user profile (name, avatar) |
| `DELETE` | `/delete` | Delete the user account |

### AI — `/api/ai`
| Method | Endpoint | Body | Description |
|--------|----------|------|-------------|
| `POST` | `/askQuery/:id` | `{ name, topic, previousExperience }` | Generate an AI-powered learning guide |

### Session Check
| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/check` | Validate the JWT token and return the authenticated user |

## 🗄️ Database Models

### `User`
```js
{
  fullName:  String (required),
  avatar:    String,
  email:     String (required),
  password:  String (required, bcrypt hashed),
  chats:     [ObjectId] → ref: CareerGuide
}
```

### `CareerGuide`
```js
{
  greetings:               String,
  prior_knowledge:         String,
  prior_knowledge_alignment: Number,
  roadmap:                 [{ step, title, description, resources[] }],
  future:                  { relevance_score, demand_growth, emerging_trends[] },
  bestBooks:               [String],
  bestCourses:             [String],
  bestYoutubeChannels:     [String],
  bestWebsites:            [String],
  goodProjects:            [{ name, description, difficulty, githubExample }],
  topCompanies:            [{ name, roles[], hiringTrends }],
  avgPackages:             { india, usa, europe },
  timestamps:              true
}
```
